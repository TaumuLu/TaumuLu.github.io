圆内接四边形

发表于 2017-06-30

前言

今天写一个效果，计算圆中的矩形宽高，随即想到圆的内接四边形，及其如何面积最大，当然内接四边形为正方形最大了，求其宽高即边长

结果

知道自己数学差，但没想到如此的差，故此写一写，以后肯定会有这方面的学习补充，数学虐我千百遍 我待数学如初恋

共式

勾股定理

aa + bb = c*c，其中c是斜边长

内接最大四边形

a = b
cc = 2 * aa，a是直角边长
c = sqrt(2) * a，sqrt(2)是计算机函数的”根号2”
