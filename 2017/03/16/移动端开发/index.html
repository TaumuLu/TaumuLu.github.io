何为移动端开发

没错做手机网页的前端工程师

开发类型

  • Native App

    也就是完全使用移动设备系统语言写的客户端
    iPhone iPad就是纯Object-C，安卓就是纯JAVA

  • Web App

    在手机浏览器里打开的网页，纯HTML+CSS+JS

  • Hybrid App

    使用原生写一个容器，使用HTML+CSS+JS来实现用户界面和交互
    开发更灵活性，原生容器可以暴露出偏底层的接口进行开发
