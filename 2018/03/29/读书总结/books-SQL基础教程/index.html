SQL基础教程

发表于 2018-03-29

第一章 数据库和SQL

DBMS种类

  • 层次型数据库
  • 关系型数据库
  • 面向对象数据库
  • XML数据库
  • 键值存储数据库

SQL语句种类

  • DDL
  • DML
  • DCL

规则

  • 分号结尾
  • 不区分大小写
  • 数字无需加引号
  • 字符串和日期常数使用单引号
  • 空格分隔
  • 只能用英文字母、数字、下划线作为数据库、表和列的名称
  • 必须以英文字母开头
  • 名称不能重复

数据类型

  • INTEGER
  • CHAR 定长字符串，空格补齐
  • VARCHAR 可变长字符串

约束

  • NULL
  • NTO NULL

命令

  • CREATE DATABASE <数据库名称>;
  • CREATE TABLE <表名> (<列名> <数据类型> <约束>, …);
    • PRIMARY KEY (列名)
